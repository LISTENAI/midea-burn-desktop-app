## 聆思模组烧录工具

### 概述

本工具允许工厂产线快速烧录 Midea_CSK6_WB01_EVB_1V0 模组的 CSK6 及 WB01 的固件，配合MES系统的情况下更可以记录产线烧录状态。

### 开发环境

- Visual Studio 2022 (17.4)
- .Net 6.0.11 (SDK 6.0.403)
- SQL Server 2022
